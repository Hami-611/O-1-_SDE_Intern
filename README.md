# Django Mini Task: Dynamic Task Page

This is a small Django project demonstrating how to create a dynamic HTML page with CSS styling and a custom URL route.

## Features

- Custom URL: `/url/task`
- `task.html` template
- `style.css` for styling
- Django view that renders dynamic data on the HTML page

## Steps to Run

1. **Install dependencies**  
   ```bash
   pip install django
2. **Run the server**
   ```bash
   python manage.py runserver
3. **Access the task page**
### Open your browser and go to:
http://localhost:8000/url/task
